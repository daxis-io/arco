# Release Notes for v1.4.0

## Highlights
- Introduces the `1.x` release line from current `main`, with planned protobuf-breaking contract changes deferred to `2.0.0`.
- Adds the `/api/v1/query` server-side SQL endpoint backed by DataFusion with Arrow IPC or JSON output.
- Lands control-plane transaction support, flow cutover wiring, and orchestration output visibility lifecycle coverage across the current API and worker stack.
