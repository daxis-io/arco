# Release Notes for v0.1.4

## Highlights
- Switched release-tag signature verification to repository-managed SSH trust roots in `.github/release-signers.allowed`.
- Removed dependency on GitHub account key registration for SBOM workflow tag verification while keeping cryptographic verification strict.
- Added deterministic `arco-<tag>.tag-verify.txt` capture in release artifacts/evidence.

## Changelog Reference
- Change log entry: [`0.1.4`](../CHANGELOG.md#014---2026-02-18)

## Verification
- Release-tag CI discipline is enforced by `.github/workflows/ci.yml` (`Release Tag Discipline` job).
- SBOM publication + retention + provenance attestation are enforced by `.github/workflows/release-sbom.yml`.
- Immutable release evidence pack is generated by `tools/collect_release_evidence.sh`.
