# Release Notes for v0.2.0

## Highlights
- Expands the catalog control-plane stack with scoped metastore mutations, catalog product APIs, Unity Catalog compatibility inventory, and tenant-visible system-table contracts.
- Lands orchestration control-plane contracts for root transactions, task-token callbacks, worker dispatch envelopes, and output visibility lifecycle coverage.
- Raises the release baseline to Rust 1.88, aligns the Buf gate to 1.70.0, and updates workspace, Python, OpenAPI, changelog, and release-note metadata for `0.2.0`.
- Fixes Tier-1 catalog snapshot retry idempotency so byte-identical immutable snapshot collisions are accepted on retry while divergent content still fails closed.
